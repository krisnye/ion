# Interpreter Value Data Model

## Symbols

    type

## Objects

    { [type]: AstClass, prop1: value1, prop2: value2, ... }

## Arrays

    [ [type]: AstType, element1, element2, element3 ]

## Integers

    { [type]: AstIntegerClass, value: 1 }

## Floats

    { [type]: AstFloatClass, value: 1.0 }

## Strings

    { [type]: AstStringClass, value: "1" }
